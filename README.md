# Delivery-return-analysis

### 📊 Project: E-Commerce Delivery Delay Analysis

Analyzed an e-commerce logistics dataset using SQL and Power BI to identify key drivers of late deliveries. Performed data cleaning, feature engineering (created `is_late` and weight-based categories), and built an interactive dashboard to uncover operational insights.

Key findings:

* Heavy-weight shipments show the highest delay rates (~50%+), making weight the strongest driver of delays
* Low and medium importance products have higher delays (~40%) compared to high-priority items (~35%)
* Combined analysis indicates that heavy, low-priority orders are the primary contributors to late deliveries, suggesting prioritization of high-importance shipments and inefficiencies in handling heavier goods

Tools used: MySQL, Power BI
